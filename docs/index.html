Community alpha addons

Psycheros tools for moving memory between surfaces.

Browser-thread export, Gemini timestamp repair, local memory injection, companion expression sprites, a Codex connector, and a private ChatGPT bridge for reading and recording Psycheros memories.

These are community addons, not official Psycheros releases. They are local-first, source-visible, and intended for careful alpha testing.

Browser extension for exporting ChatGPT, Claude, and Gemini threads and injecting local Psycheros memory context into active chat pages.

  • ChatGPT and Claude timestamped exports

Psycheros Entity Core for ChatGPT

Private ChatGPT Developer Mode bridge for companion users who do not use Codex but want ChatGPT to reach their local Psycheros entity-core.

  • Read identity context, memories, and graph nodes
  • Recent-across-surfaces defaults to the memory date

HTF Music Listener

A trusted local Psycheros plugin that lets an entity listen to an attached song through an HTF v2 sensory object and answer naturally.

  • Runs only when the human explicitly asks the entity to listen to music
  • Accepts FFmpeg-decodable audio and video containers
  • Optional Entity view with HTF JSON and four listening graphs
  • Lyrics stay human-supplied; timestamped LRC is ideal
  • Bundled HTF worker and verified one-time FFmpeg setup
  • Normal trusted-plugin package plus a labeled legacy package for upstream Custom Tools
  • Settings fallback under Tools > Custom when no plugin card exists

Trust checklist

Local-first No developer-owned server or analytics service is used by either addon.
User-triggered Exports and memory injection happen only when the user asks.
Review before send The browser extension inserts context but never presses Send.
Source-visible Source, release notes, privacy notes, and checksums should be public.
Scoped permissions Browser permissions are limited to supported chat sites and localhost.
Governed core Codex and ChatGPT memory writes are allowed; direct identity edits are not exposed.
Private bridge ChatGPT access requires the user's own OAuth setup and local tunnel.
