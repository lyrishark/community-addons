# Historical Psycheros source packages

These packages are preserved through Git tags, releases, and the final 0.9.2
source snapshot. They replaced host files and are not compatible with the
Psycheros 0.10 plugin manager.

Do not copy any of these source trees over Psycheros 0.10.

The historical links are immutable references, not installation
