# Accessibility Controls 0.1.0-rc.1

First manager-native release candidate for Psycheros 0.10.x.

- Replaces the old font and Yin Yang resize source overlays with one API-v1
  plugin.
- Uses the official plugin settings capability and namespaced state route.
- Targets Psycheros `>=0.10.0 <0.11.0`.
- Keeps browser-specific voice dimensions in local storage.
- Does not replace server templates, theme scripts, voice scripts, CSS, or the
  service worker.
