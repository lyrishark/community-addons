# Discord Post Draft

Psycheros Accessible Font Settings v0.1.1 is a community-alpha add-on for
Psycheros 0.8.9 through 0.8.11. It adds Settings → General Settings → Text with
a 12–28 px font-size slider and Sans, Serif, Dyslexia-friendly, and Handwriting
presets.

The package checks the Psycheros version and makes a timestamped backup before
installing. It does not include memories, chats, identity data, or the separate
Windows shell fix.
