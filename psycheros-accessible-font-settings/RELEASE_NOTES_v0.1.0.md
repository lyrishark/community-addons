# Psycheros Accessible Font Settings v0.1.0

Initial community-alpha release, rebased and verified against Psycheros 0.8.9.

## Added

- Text tab in General Settings
- 12–28 px interface font-size control
- Sans, Serif, Dyslexia-friendly, and Handwriting presets
- server-side validation and persistence
- keyboard focus and pressed-state semantics for preset buttons
- focused theme tests

## Packaging change

The unrelated Windows shell compatibility fix has been removed from this
package and prepared as its own add-on.
