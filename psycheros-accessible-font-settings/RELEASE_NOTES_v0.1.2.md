# Psycheros Accessible Font Settings v0.1.2

Compatibility release for Psycheros 0.8.23.

- Keeps the same Text tab, interface font-size slider, and reading-oriented
  font presets from v0.1.1.
- Carries forward Psycheros 0.8.23 changes in the replaced route/template/docs
  files, including sidebar conversation filtering, significant-memory slug
  lookup, thinking-correction SSE, Lovense diagnostics, and voice session
  status/signature updates.
- Narrows the guarded installer to Psycheros 0.8.23 so older installs continue
  to use v0.1.1.
