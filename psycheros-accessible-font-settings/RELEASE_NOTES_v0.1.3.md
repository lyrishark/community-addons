# Psycheros Accessible Font Settings v0.1.3

Compatibility patch for Psycheros 0.8.23.

## Changes

- Expands the font preset stacks with OS-aware fallbacks for Windows, macOS/iOS,
  Android, and Linux.
- Keeps the add-on font-only: no bundled font downloads or licensing changes.
- Updates the General Settings preview buttons so their sample fonts match the
  actual applied preset stacks.
- Adds test coverage for the cross-platform fallback entries.
