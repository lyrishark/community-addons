# Accessible Font Settings 0.2.0

Compatibility release for **Psycheros 0.9.2**.

- Rebased the Text settings tab, size controls, font presets, and platform-aware
  fallback stacks onto pristine upstream 0.9.2.
- Preserved current upstream theme and settings behavior.
- Passed type checking, browser-script syntax checking, and all 11 focused
  theme tests.
- The installer accepts only Psycheros 0.9.2.
