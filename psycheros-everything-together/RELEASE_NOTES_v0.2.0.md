# Everything Together 0.2.0

Stable compatibility release for **Psycheros 0.9.2**.

- Rebased and composed More Uploads, Voice Text Resize, Accessible Font
  Settings, Windows Shell Fix, Screen Presence Alpha, and Expression Sprites on
  pristine upstream 0.9.2.
- Preserves current upstream attachment IDs, transient vision images,
  reasoning, plugins, image generation, voice behavior, and custom sprite
  persistence.
- The Windows x64 release bundles HTF Music Listener 0.1.3 legacy; other
  platforms receive the source features without claiming the compiled listener.
- Removes the unrelated missing-response regeneration button, voice-started
  auto-title behavior, and queued typed-turn draining from both payload and
  documentation.
- Passed type checking, browser-script syntax checking, and all 67 focused
  feature tests.
- The installer accepts only Psycheros 0.9.2.
