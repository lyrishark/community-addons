# Psycheros Expression Sprites Beta v0.1.0

Initial beta package for Psycheros 0.8.21.

## Included

- Live expression-state events for entity turns.
- Settings > Vision > Expressions.
- SillyTavern-style sprite ZIP import.
- Per-expression sprite upload and delete.
- Fake checkerboard cleanup for common generated "transparent" backgrounds.
- Label, closest-sprite, and hidden fallbacks for missing sprites.
- Desktop and mobile visual-novel-style sprite display.
- Focused expression classifier, sprite import, and checkerboard cleanup tests.

## Not included

- Sprite images.
- Screen sharing.
- Memory writes for expression state.

## Install

Run `install.ps1` from this folder. The installer checks for Psycheros 0.8.21,
backs up replaced files, copies the add-on files, and prints next steps.
