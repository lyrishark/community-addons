# Psycheros More Uploads + Voice Text Resize 0.1.1

This combo now includes the full music-upload path from More Uploads 0.1.1 while
preserving its resizable Yin Yang typed voice composer.

- Common music formats are accepted in chat and Yin Yang typed voice.
- Large files stream to disk with a 512 MB ceiling.
- Audio persists as `USER_AUDIO` and reopens with browser audio controls.
- Install this package before HTF Music Listener 0.1.2 legacy on stock Psycheros.
