Compatibility release for **Psycheros 0.9.2**.

- Rebased multiple image, document, and music uploads onto current chat and
  voice surfaces.

  plugins, and image-generation behavior.

- Passed type checking, browser-script syntax checking, and all 6 focused
  attachment tests.
- The installer accepts only Psycheros 0.9.2.
